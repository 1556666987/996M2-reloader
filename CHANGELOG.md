# 更新日志

本项目采用规范化版本号：`主版本.次版本.修订版本.构建版本`，标签格式为 `v主版本.次版本.修订版本.构建版本`。

## 版本号规则

- 主版本：发生不兼容的重大变化时递增。
- 次版本：新增功能且保持兼容时递增。
- 修订版本：修复 Bug、优化稳定性时递增。
- 构建版本：文档、配置、资源文件、打包或内部维护变更时递增。

递增规则：

- 主版本递增时，次版本、修订版本、构建版本归零。
- 次版本递增时，修订版本、构建版本归零。
- 修订版本递增时，构建版本归零。
- 构建版本独立递增。

示例：

```text
v1.0.0.0  初始稳定版
v1.0.1.0  Bug 修复版
v1.1.0.0  兼容性新功能版
v2.0.0.0  重大不兼容版本
v1.0.1.1  资源、文档或构建维护版
```

## v1.0.1.0 - 2026-05-24

### 修复

- 修复双击系统托盘图标无法唤出主界面的问题。
- 显示主界面时恢复窗口正常状态，并短暂置顶，降低被其他窗口遮挡的概率。

### 变更

- 托盘图标改为使用项目内置 `icon.ico`。
- 移除对本机固定路径 `D:\源码\1.ico` 的依赖。
- 新增 `BASE_DIR`，统一基于程序所在目录定位配置文件和图标资源。

### 文档

- 新增 `README.md`，补充项目用途、功能特性、依赖安装、使用方法、配置说明和回滚说明。

## v1.0.0.0 - 2026-05-24

### 初始稳定版

- 自动查找标题包含 `996引擎` 和 `KUAFU` 的 M2Server 窗口。
- 验证目标进程为 `M2Server.exe`。
- 自动读取“控制 -> 重新加载”菜单下的子项。
- 支持勾选需要执行的重载项目。
- 支持按 `Ctrl+S` 批量执行已勾选项目。
- 支持全选、取消全选、刷新菜单。
- 支持修改 M2Server 日志窗口字体和字号。
- 自动保存字体配置和勾选项到 `config.json`。
- 启动后默认隐藏到系统托盘。
- 关闭窗口时最小化到托盘，托盘菜单支持显示和退出。
- 清理测试脚本、旧版主程序和备份目录。
